Access Radicle

Desktop

A desktop client for Radicle.

Check out the Guide to Getting Started.

Web

A web client for Radicle.

Radicle Interface is an open source web-based client enabling Users to access Radicle-Link and the Radicle’s smart contract system.

Etherscan is a Block Explorer and Analytics Platform for Ethereum, a decentralized smart contracts platform. The contract to register Radicle names can be found here and the contract to create Radicle orgs can be found here.

ethBuild is an educational sandbox for web3, including drag-and-drop programming and open-source building blocks. With ethBuild, you can call a function on an Ethereum contract (like those to register Radicle names and to create Radicle orgs) by sending it a transaction with the function args in the data.

MyEtherWallet is a free, client-side interface helping you interact with the Ethereum blockchain. Their easy-to-use, open-source platform allows you to generate wallets, interact with [Radicle] smart contracts, and more.

The web3.js Library makes it easy to interact with smart contracts on the Ethereum blockchain, including Radicle smart contracts.

The ethers.js Library aims to be a complete and compact library for interacting with the Ethereum Blockchain and its ecosystem, including Radicle smart contracts.
